Evolução dos Computadores

Voltar

Na primeira parte de nossa aula, conhecemos sobre as gerações de dispositivos que passamos, estando hoje na quarta geração ou quinta geração. A primeira geração marcada pelas máquinas de cálculos complexos para ajudar na guerra, o que não só ajudou na guerra como contribuiu tanto para os dias atuais; na segunda geração evoluímos graças aos transistores, reduzindo o tamanho da máquina consideravelmente; Na terceira geração evoluímos dos transistores para um circuito integrado contendo transistor, capacitores e resistores, reduzindo mais ainda o tamanho da máquina e a tornando mais precisa. Na quarta geração a partir de 1971 temos os nossos primeiros computadores pessoais, a Apple nos apresentou o mouse, que foi uma invenção sensacional que usamos até os dias atuais, e a cada dia estamos evoluindo mais passando talvez para a quinta geração marcada pela inteligencia artificial.

Primeiro contato com a Internet

Voltar

Como podemos imaginar, a maioria das coisas que temos em nosso dia a dia hoje, foram provenientes das guerras, das necessidades exigidas na guerra, uma dessas invenções foi a internet, mas para entendermos isso, temos que voltar no passado e ver como foi sua evolução. Começando pelo telégrafo, onde usavam para se comunicar, após, tivemos a guerra fria na década de 1950, com todos estes avanços, tivemos um nome muito importante nesse contexto, Joseph Carl Robnett Licklider, onde o mesmo teorizou sobre uma rede onde todos poderiam obter contato através dela.

Primeira conexão Estabelecida

Voltar

A comunicação através de rede era uma exclusividade maior do exercito, não tendo acesso ao público da maneira como temos atualmente, contudo foram se discutindo sobre este meio de comunicação até que se obteve a primeira conexão estabelecida de internet, realizada em 29/10/1969, a palavra "LOGIN" foi enviada de um lado a outro de UCLA a SRI, porém só chegou "LO".

TCP/IP

Voltar

Após muitos estudos Vinton Cerf e Robert Kahn criaram o TCP/IP onde poderiamos criar uma conexão entre computadores através de uma rede e em 1989 tivemos outro grande nome, Tim Berners-Lee*, qual viu uma oportunidade de unir hipertexto com TCP/IP para criar a World Wide Web (WWW)

Tim Berners-Lee, criador da World Wide Web (WWW)

Um breve resumo sobre Client x Server

Voltar

O client tem o "papel" de consumir informações enviadas pelo server, como por exemplo no client rodamos arquivos estáticos, sendo HTML, CSS, Javascript, Imagens. No server, a função principal é servir de apoio para os clients que iram acessar os arquivos que são transmitidos pelo server, existem vários tipos de server, sendo: segurança, streaming de dados, e-mail, web, banco de dados.

Tags o que são?

Voltar

São basicamente comandos inseridos para definir o que você deseja obter no resultado final da codificação.

Atributos de Tag

Voltar

São detalhes que pode-se obter junto da tag que está utilizando, como exemplo utilizando a tag strong, usando dentro dela o atributo style para mudar a cor da fonte.

Tags em Textos

Voltar

As tags mais utilizadas para texto são 'p' para paragrafos, h1 a h6 para titulos, estas são as tags mais basicas básicas para organizar o texto em seu site.

Listas ordenadas e não ordenadas

Voltar

As listas ordenadas também são tags sendo 'ol', usadas para fazer uma lista númerica de dados, enumerados automaticamente de 1 até a quantidade desejada, utilizamos junto a tag 'li' como complemento para criar os nomes que estarão na lista. Lista não ordenadas são basicamente a mesma forma da descrita acima, a diferença é que não é uma lista enumerada numéricamente, somente com pontos ao lado para identificar a lista.
